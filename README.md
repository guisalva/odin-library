# Odin - Library

![Status Badge](https://img.shields.io/badge/status-concluded-brightgreen?style=for-the-badge)

This project was built to practice JS Objects from the **JavaScript Course** of [The Odin Project](https://www.theodinproject.com).

It is a virtual Library with create, delete and toggle read status operations.

## 🛠️ Tech Stack

- HTML
- CSS
- JS

## 🚀 Installation and Usage

Clone the repository:

```bash
git clone https://github.com/guisalva/odin-library.git

cd odin-library
```

Run the project:

1. Open `index.html` in your preferred browser;
2. Or use an extension like Live Server for a better development experience;
